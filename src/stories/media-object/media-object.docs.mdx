# MediaObject

The `MediaObject` component is a flexible layout pattern that combines media (icon or image) with content (title and description). It's perfect for displaying user profiles, product cards, notifications, and any content that needs a media element alongside text.

## Features

- **Flexible Media**: Support for both icons and images
- **Consistent Layout**: Standardized media + content pattern
- **Interactive**: Optional click handling for interactive elements
- **Spacing Control**: Comprehensive margin utilities
- **Accessibility**: Proper semantic structure and alt text support
- **Responsive**: Adapts to different screen sizes

## Usage

```jsx
import { MediaObject } from "cleanplate";

const App = () => {
  return (
    <MediaObject
      mediaIcon="user"
      title="John Doe"
      description="Senior Developer at Tech Corp"
    />
  );
};
```

## Interactive Examples

### Clickable Media Objects
```jsx
const InteractiveMediaObject = () => {
  const [selected, setSelected] = useState(false);
  
  return (
    <div 
      style={{ 
        border: selected ? "2px solid var(--primary-brand)" : "1px solid var(--gray-light)",
        borderRadius: "var(--radius-medium)",
        padding: "var(--space-3)",
        cursor: "pointer",

        mediaImage="https://images.unsplash.com/photo-1472099645785-5658abf4ff4e?w=150&h=150&fit=crop&crop=face"
        title="John Doe"
        description="Senior Developer • Click to select"

  );
};
```

## Best Practices

1. **Use Appropriate Media**: Choose icons for functional elements and images for visual content
   ```jsx
   // Good: Icon for settings
   <MediaObject mediaIcon="settings" title="Settings" description="..." />
   
   // Good: Image for user profile
   <MediaObject mediaImage="avatar.jpg" title="John Doe" description="..." />
   ```

2. **Provide Alt Text**: Always include meaningful alt text for images
   ```jsx
   // Good: Descriptive alt text
   <MediaObject 
     mediaImage="user-avatar.jpg" 
     title="John Doe" 
     description="Senior Developer"
   />
   ```

3. **Consistent Spacing**: Use design system spacing tokens
   ```jsx
   // Good: Design system spacing
   <MediaObject margin="m-3" title="..." description="..." />
   
   // Avoid: Custom spacing
   <MediaObject style={{ margin: "12px" }} title="..." description="..." />
   ```

4. **Meaningful Descriptions**: Provide helpful context in descriptions
   ```jsx
   // Good: Informative description
   <MediaObject 
     title="John Doe" 
     description="Senior Developer • john.doe@example.com" 
   />
   
   // Avoid: Vague description
   <MediaObject 
     title="John Doe" 
     description="User" 
   />
   ```

## Accessibility

- **Semantic HTML**: Uses proper heading hierarchy (h3 for titles)
- **Alt Text**: Images include alt text based on the title prop
- **Keyboard Navigation**: Interactive media objects should be keyboard accessible
- **Screen Readers**: Proper structure for screen reader announcements
- **Focus Management**: Clickable media objects should have visible focus states
