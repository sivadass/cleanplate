# Statistic

`Statistic` displays a labeled numeric metric for dashboards and summary tiles — from a simple KPI (`title` + `value`) to a full card with icon, progress bar, description, and footer badge.

Use it for **KPI highlights**. For inline status labels use `Badge`; for general copy use `Typography`; for tabular data use `Table`.

> **`Statistic.Timer` (countdown/countup) is planned** — not available yet.

## When to use

| Use `Statistic` | Use something else |
| --- | --- |
| Dashboard totals, collection rates, overdue amounts | `Table` for row-level data |
| Card-style metric with icon + subtext | `Typography` alone for prose |
| Progress + percentage badge under a count | `Badge` alone for a status chip |
| Loading KPI while fetching | `Spinner` alone (Statistic composes it) |

## Anatomy

**Plain** (`variant="plain"`):

```
title
prefix + value + suffix
```

**Card** (`variant="card"`):

```
[icon] title          ← header row
prefix + value + suffix
[progress bar]        ← optional
description           ← optional subtext
label          badge  ← optional footer row
```

Set `icon` on a plain statistic to get the header row without card chrome.

## Features

- **Value:** `string` (verbatim) or `number` (grouping + optional `precision`; non-finite → `—`)
- **Prefix / suffix:** `ReactNode` slots before/after the value; hidden while `loading`
- **Card variant:** `variant="card"` — border, padding, tinted surface when a `tone` is set
- **Tones:** `neutral`, `success`, `warning`, `danger`, `muted` — icon accent, value color, card surface, and default progress/badge styling
- **Icon / description / progress / footer:** optional card slots; hidden while `loading`
- **Sizes:** `small`, `medium`, `large` — typographic scale for title and value
- **Loading:** title and icon stay visible; value row shows `Spinner`
- **Spacing:** `margin` uses the suffix API (e.g. `"b-2"` → m-b-2)
- **Customization:** root `className` + documented `cp-` slot classes (no per-slot props)

## Usage

```jsx
import { Statistic } from "cleanplate";

const App = () => (
  <Statistic title="Active Users" value={112893} size="medium" />
);
```

Interactive controls live in **atoms/Statistic/Playground**.

## Tones

`tone` drives icon color, value emphasis, card background (with `variant="card"`), and the default `ProgressBar` / `Badge` variants when you omit `progress.variant` or `footer.badgeVariant`.

| Tone | Use for | Card surface | Value | Default progress | Default badge |
| --- | --- | --- | --- | --- | --- |
| `neutral` | Totals, informational | White | Default text | `default` | `default` |
| `success` | Completed, collected | Green tint | Default text | `success` | `success` |
| `warning` | Attention needed | Orange tint | Default text | `warning` | `warning` |
| `danger` | Critical, overdue | Red tint | **Red** | `error` | `error` |
| `muted` | Zero-state, waived | Gray tint | **Muted** | `info` | `info` |

```jsx
<Statistic variant="card" tone="neutral" title="Total billed" value={9000} prefix="₹" />
<Statistic variant="card" tone="success" title="Collected" value={3375} progress={{ value: 38 }} />
<Statistic variant="card" tone="danger" title="Overdue (>30d)" value={0} prefix="₹" />
<Statistic variant="card" tone="muted" title="Waived" value={0} progress={{ value: 0 }} />
```

## Examples

### Basic and precision

```jsx
<Statistic title="Active Users" value={112893} />
<Statistic title="Account Balance (CNY)" value={112893} precision={2} />
```

### Prefix, suffix, and string values

```jsx
import { Statistic, Icon } from "cleanplate";

<Statistic
  title="Feedback"
  value={1128}
  prefix={<Icon name="thumb_up" size="small" />}
/>
<Statistic title="Unmerged" value={93} suffix="/ 100" />
<Statistic title="Revenue" value="¥1.2M" />
```

Pass a **string** `value` to skip numeric formatting (`precision`, separators ignored).

### Card with icon and description

```jsx
import { Statistic, Icon } from "cleanplate";

<Statistic
  variant="card"
  tone="neutral"
  icon={<Icon name="group" size="small" />}
  title="Total billed"
  value={9000}
  prefix="₹"
  description="8 active members"
/>
```

### Progress and footer badge

`progress.value` is 0–100. Tone picks the bar color unless you set `progress.variant`.

```jsx
<Statistic
  variant="card"
  tone="success"
  icon={<Icon name="check_circle" size="small" />}
  title="Paid dues"
  value={3}
  progress={{ value: 38, size: "small" }}
  footer={{ label: "of 8 total", badge: "38%" }}
/>
```

String `footer.badge` renders `Badge` with the tone default (or `footer.badgeVariant` override). Pass a `ReactNode` for custom badge content.

There is no built-in grid — compose with `Container` and `Typography` for section headings:

        variant="card"
        tone="neutral"
        icon={<Icon name="group" size="small" />}
        title="Total billed"
        value={9000}
        prefix="₹"
        description="8 active members"
      />

### Loading

Title and icon remain visible. The value row shows a `Spinner`; `prefix`, `suffix`, `progress`, `description`, and `footer` are hidden. The content row sets `aria-busy={true}`.

```jsx
<Statistic title="Active Users" value={112893} loading />
```

## Value formatting

| `value` type | Behavior |
| --- | --- |
| `number` | Grouped with `groupSeparator` (default `,`); `precision` fixes decimals; `NaN` / `±Infinity` → `—` |
| `string` | Rendered verbatim; separators and `precision` ignored |
| omitted | No value text; title/layout still render |

## Spacing (margin)

`margin` accepts the **spacing suffix**. The component adds the `m-` prefix internally.

```jsx
<Statistic title="Active Users" value={112893} margin="0" />
<Statistic title="Active Users" value={112893} margin="b-2" />
```
