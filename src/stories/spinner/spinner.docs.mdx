# Spinner

The `Spinner` component is a lightweight and customizable loading indicator, supporting various sizes, themes, and spacing options. It seamlessly conveys progress or activity status within different layouts and design systems.

### Usage

```
import { Spinner } from "cleanplate";

const App = () => {
  return (
    <Spinner
        size="medium"
        className="my-custom-icon"
        variant="light"
    />
  );
}
```
