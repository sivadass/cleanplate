# Header

The `Header` component provides a responsive navigation header with logo, menu items, and customizable left, center, and right slots. Use it for app navigation with a responsive mobile menu that slides in from the left. Supports sizes, variants, and margin spacing.

## Features

- **Logo:** `logoUrl` for default left content, or `headerLeft` for custom content
- **Menu items:** `menuItems` (label, value, optional icon) drive center nav and mobile menu
- **Slots:** `headerLeft`, `headerCenter`, `headerRight` for custom content
- **Active state:** `activeMenuItem` and `onMenuItemClick` for navigation
- **Sizes:** `small`, `medium`, `large`
- **Variants:** `light`, `dark`
- **Spacing:** Margin uses the suffix API (e.g. `"0"` → m-0)
- **Mobile menu:** Hamburger trigger, slide-in menu with close button

## Usage

```jsx
import { useState } from "react";
import { Header, Avatar } from "cleanplate";

const MENU_ITEMS = [
  { label: "Dashboard", value: "/", icon: "speed" },
  { label: "Projects", value: "/projects", icon: "receipt_long" },
];

const AppHeader = () => {
  const [activeMenuItem, setActiveMenuItem] = useState("/");
  return (
    <Header
      logoUrl="/logo.svg"
      menuItems={MENU_ITEMS}
      activeMenuItem={activeMenuItem}
      onMenuItemClick={(item) => setActiveMenuItem(item.value)}
      headerRight={<Avatar name="John" />}
    />
  );
};
```

## Spacing (margin)

`margin` accepts the **spacing suffix**. The component adds the `m-` prefix internally (e.g. `"0"` → m-0, `"b-2"` → m-b-2). Use a single string or array of suffixes.

```jsx
<Header menuItems={items} margin="0" />
<Header menuItems={items} margin={["b-2"]} />
```

## Behavior

- **menuItems:** Required; each item has `label`, `value`, optional `icon` (Material icon name).
- **headerLeft / headerCenter / headerRight:** When provided, replace the default logo, MenuList, or right slot.
- **Mobile:** Below 1024px, center nav hides; hamburger shows. Click opens slide-in menu (Animated fade-in-left).
- **onMenuItemClick:** Called with the clicked item; mobile menu closes on click.
- **DOM:** Root div with wrapper; left/center/right sections; mobile menu panel with MenuList.
