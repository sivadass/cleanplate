# Footer

The `Footer` component provides a customizable footer with copyright text, optional powered-by link, and custom content via children. Use it for app footers with branding, legal links, or column layouts. Supports sizes, variants, and margin spacing.

## Features

- **Copyright:** `brandName` in "© [year] [brandName]. All rights reserved." (year is current year)
- **Powered-by link:** `poweredByLabel` and `poweredByLink` for optional attribution
- **Custom content:** `children` rendered above the copyright line
- **Sizes:** `small`, `medium`, `large`
- **Variants:** `light`, `dark`
- **Spacing:** Margin uses the suffix API (e.g. `"0"` → m-0)

## Usage

```jsx
import { Footer } from "cleanplate";

const AppFooter = () => (
  <Footer
    brandName="Acme Inc"
    poweredByLabel="Powered by Sivadass"
    poweredByLink="https://sivadass.in"
  />
);
```

## Examples

### Basic

```jsx
<Footer brandName="Acme Inc" />
```

### With powered-by link

```jsx
<Footer
  brandName="Acme Inc"
  poweredByLabel="Powered by Sivadass"
  poweredByLink="https://sivadass.in"
/>
```

### Variants

```jsx
<Footer brandName="Acme Inc" variant="light" />
<Footer brandName="Acme Inc" variant="dark" />
```

### Sizes

```jsx
<Footer brandName="Acme Inc" size="small" />
<Footer brandName="Acme Inc" size="medium" />
<Footer brandName="Acme Inc" size="large" />
```

### With custom content (columns)

```jsx
<Footer brandName="Acme Inc">
  <Container display="flex" gap="4">
    <Container width="quarter">
      <Typography variant="h6" margin={["0", "b-2"]}>The Company</Typography>
      <ul>
        <li><a href="/contact">Contact Us</a></li>
        <li><a href="/about">About Us</a></li>
      </ul>
    </Container>
    <Container width="quarter">
      <Typography variant="h6" margin={["0", "b-2"]}>Policies</Typography>
      <ul>
        <li><a href="/terms">Terms</a></li>
        <li><a href="/privacy">Privacy</a></li>
      </ul>
    </Container>
  </Container>
</Footer>
```

## Spacing (margin)

`margin` accepts the **spacing suffix**. The component adds the `m-` prefix internally (e.g. `"0"` → m-0, `"b-2"` → m-b-2). Use a single string or array of suffixes.

```jsx
<Footer brandName="Acme Inc" margin="0" />
<Footer brandName="Acme Inc" margin={["b-2"]} />
```

## Behavior

- **Copyright:** Uses current year and `brandName`. Powered-by link shows only when both `poweredByLabel` and `poweredByLink` are set.
- **children:** Rendered in a wrapper div above the copyright line.
- **DOM:** Native `footer` element. Typography for copyright; link has `rel="noopener noreferrer"` when `target="_blank"`.
