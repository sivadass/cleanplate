# Footer

The `Footer` component serves as a customizable and responsive footer element, allowing for branding, copyright details, and optional powered-by links. It supports custom content, adaptable sizes, themes, and spacing options to seamlessly integrate with various design requirements.

### Usage

```
import { Footer } from "cleanplate";

const AppFooter = () => {
  return (
    <Footer
        brandName="Acme Inc."
        poweredByLabel="Google"
        poweredByLink="https://google.com"
    />
  );
}
```
