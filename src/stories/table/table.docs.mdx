# Table

The `Table` component provides a clean, accessible way to organize and display structured data. Designed with flexibility in mind, it supports a variety of use cases, from simple tabular layouts to complex data-heavy views.

### Usage

```
import { Table } from "cleanplate";

const App = () => {
  return (
    <Table
      columns={[
        {
          id: "full_name",
          title: "Full Name",
          widthPercentage: "60%",
        },
        {
          id: "email_address",
          title: "Email Address",
          widthPercentage: "40%",
          textAlign: "right"
        }
      ]}
      data={[
        {
          full_name: "John Doe",
          email_address: "john@doe.com",
        },
        {
          full_name: "Jane Doe",
          email_address: "jane@doe.com",
        }
      ]}
    />
  );
}
```
