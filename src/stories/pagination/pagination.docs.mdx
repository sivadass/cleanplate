# Pagination

The `Pagination` component enables users to navigate through large sets of content by breaking them into manageable pages, ensuring clarity and ease of browsing.

### Usage

```
import { Pagination } from "cleanplate";

const App = () => {
  const [currentPage, setCurrentPage] = useState(1);
  const handlePageChange = (page, rPerPage) => {
    setCurrentPage(page);
  };
  return (
    <Pagination
      variant="default"
      totalItem={1200}
      currentPage={currentPage}
      onPageChange={(page, rPerPage) => handlePageChange(page, rPerPage)}
    />
  );
}
```
