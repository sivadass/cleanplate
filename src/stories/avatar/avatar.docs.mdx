# Avatar

The `Avatar` component is a simple and adaptable UI element for displaying user initials, supporting customizable sizes, spacing, and click events. It seamlessly integrates into layouts and enhances user interfaces with a personalized touch.

### Usage

```
import { Avatar } from "cleanplate";

const App = () => {
  return (
    <Avatar
        name="John Doe"
        size="medium"
        className="my-custom-avatar"
    />
  );
}
```
