# ProgressBar

The `ProgressBar` component is a versatile and customizable visual indicator for tracking progress, supporting various sizes, themes, and dynamic updates. It seamlessly integrates into user interfaces to represent completion status effectively.

### Usage

```
import { ProgressBar } from "cleanplate";

const App = () => {
  return (
    <ProgressBar
        size="medium"
        className="my-custom-progress-bar"
        variant="success"
    />
  );
}
```
